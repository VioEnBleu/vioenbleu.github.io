NebulaNode
The Private Server

Private, powerfull and secure are his names

NebulaNode is a simple but powerfull server running window 11 (tiny) and Parsec with a virtual display.
32Go of ram, an i7 and an RX580, a small cost beast.

Key Features Tech Stack

Key Features

Privacy And Security

NebulaNode is a fully protected server by 3 layers of security and only one option to connect to the UI, Parsec.

Colaboration Project

NebulaNode is acutally being used by four members, me included. Each know what to do and work to maintain it.

Privacy Focused

Raid F1 and 2To of strage, enought for the four of us and some usefull data like the whole wikipedia website...

Small But Powerfull

As big as three stacked laptops, his small form factor for his performances are worth the few watts he's using.

Real Life Uses

Used to run scripts, apps, servers and more, NebulaNode is a cornerstone of many projets of the group.

Development Highlights

Hardware

NebulaNode is a hundred percent made of recycled parts of all kinds except for the PSU and the SSD. The i7 and the RX580 are the real kick on this build. 32Go on top of that to complete this multitask beast.

Windows but lighter

NebulaNode is using Tiny 11, a lighter version of windows 11 with bloatware removed and local account.

Online

A smartphone, a bit of connexion, parsec and you are good to go! Dev, play and remotely control everything.
